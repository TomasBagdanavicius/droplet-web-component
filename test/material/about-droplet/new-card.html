You have now opened a new card!

The content for this new card was fetched over the network from the resource location that the link that you have just clicked is pointing to.

All links (unless chosen otherwise) in Droplet's content area will be auto-captured to be processed internally by this web plugin.

In this default setup a new card is represented by a breadcrumb item in the top clickable breadcrumb bar.

We are sure you noticed the CSS slide effect that was used to pull the new content into the view. We call it the film-track effect. Droplet supports 3 transition effects that are fully CSS driven:

  1. Film-Track
  2. Slide-Over
  3. Cross-Fade
